Regular API function

simSetJointPose / sim.setJointPose

Description Sets the intrinsic orientation of a spherical joint object. This function should not be used with non-spherical joints (use sim.setJointPosition instead). See also sim.getJointPose.
C synopsis simInt simSetJointPose(simInt objectHandle,const simFloat* pose)
C parameters
objectHandle: handle of the joint object
pose: pointer to the pose (x,y,z,qx,qy,qz). Only the quaternion part is taken into account
C return value
-1 if operation was not successful
