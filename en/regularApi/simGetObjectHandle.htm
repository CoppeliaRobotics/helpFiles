Regular API function

sim.getObjectHandle

Description Retrieves an object handle based on its path and alias. See also sim.isHandle and sim.getObjectUniqueIdentifier.
C synopsis From C/C++, call simGetObjectHandleEx instead.
C parameters
C return value
Lua synopsis int objectHandle=sim.getObjectHandle(string objectPath,table options={})
Lua parameters
objectPath: the path and alias of the object to find. See the section on accessing scene objects for details.
options: optional table indicating how the object should be searched. {noError=true} will generate a silent error if an object was not found. {index=n} will fetch the n-th object (zero based index) that satisfies the specified objectPath. {proxy=proxyObjectHandle} allows to search the hierarchy tree of the specified proxy object.
Lua return values
objectHandle: The handle of the object, or -1 in case the object was not found.
