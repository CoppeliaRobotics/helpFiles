Regular API function

sim.getScriptHandle

Description Retrieves the handle of a script.
C synopsis From C/C++, call simGetScriptHandleEx instead.
C parameters
C return value
Lua synopsis int scriptHandle=sim.getScriptHandle(int scriptType,int objectHandle=-1 / string scriptName='')
Lua parameters
scriptType: the type of the script.
objectHandle: the object that this script is associated with (in case of child- or customization scripts. Can be omitted.
scriptName: name of the script, if this script is an add-on. Can be omitted.
Lua return values
handle of the script, or -1 if the script does not exist.
