Regular API function

sim.packCbor

Description Packs an argument as a CBOR-encoded buffer. Strings with an @:txt: suffix will be forcibly packed as text, and strings with an @:dat: suffix will be forcibly packed as byte buffer. See also the other packing/unpacking functions.
