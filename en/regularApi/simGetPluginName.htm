Regular API function

simGetPluginName / sim.getPluginName

Description Retrieves a plugin name based on an index
C/C++
synopsis 		char* simGetPluginName(int index)
C/C++
parameters
index: index to a plugin. To list-up all plugin names, start with index=0 and increment index until return value is nullptr
C/C++
return value
Name of the plugin or nullptr if no plugin is available at index position, or in case of an error. The user is in charge of destroying the returned name with simReleaseBuffer
Lua
synopsis 		string pluginName=sim.getPluginName(int index)
Lua
parameters
index: index to a plugin. To list-up all plugin names, start with index=0 and increment index until return value is nil
Lua
return values
pluginName: name of the plugin or nil if no plugin is available at that index position.
Python
synopsis		 string pluginName=sim.getPluginName(int index)
