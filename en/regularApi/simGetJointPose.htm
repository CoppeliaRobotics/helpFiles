Regular API function

simGetJointPose / sim.getJointPose

Description Retrieves the intrinsic pose of a joint (the transformation caused by the joint movement). See also sim.setJointPose.
C synopsis simInt simGetJointPose(simInt objectHandle,simFloat* pose)
C parameters
objectHandle: handle of the joint
pose: pointer to the pose (x,y,z,qx,qy,qz,qw)
C return value
-1 if operation was not successful
Lua synopsis table[7] pose=sim.getJointPose(int objectHandle)
Lua parameters
objectHandle: handle of the joint
Lua return values
pose: the pose (x,y,z,qx,qy,qz)
