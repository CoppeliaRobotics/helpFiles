Regular API function

sim.closeTextEditor

Description Closes a non-modal text edition window previously opened with sim.openTextEditor. Should only be called from the same script that opened the text edition window.

-
C parameters
-
C return value
-
Lua synopsis
number result=sim.closeTextEditor(number handle,boolean ignoreCallback)
Lua parameters
handle: the handle of the non-modal text editor window.
ignoreCallback: if true, the callback function specified when opening the non-modal text editor window will be ignored, otherwise the callback function will be called if the specified window was not closed yet.
Lua return values
result: 0 if the text editor window does not exist or was already closed, otherwise 1
