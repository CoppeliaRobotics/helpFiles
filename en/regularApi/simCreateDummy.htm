simCreateDummy / sim.createDummy

Description Creates a dummy.
C synopsis simInt simCreateDummy(simFloat size,const simFloat* color)
C parameters
size: the dummy size
color: pointer to 3 values representing the dummy color. Can be nullptr for default values
C return value
-1 if operation was not successful, otherwise the handle of the dummy
